osmoscraft

osmoscraft is a growing knowledge toolbox that pushes you to discover, ingest, organize, connect, generate, and share knowledge.
In the short term, the toolbox focuses on secondary tasks: reading articles, bookmarking websites, and taking notes.
In the long term, the toolbox creates a personalized learning environment for discovering new knowledge and synthesizing new insights.

Learning

The project aims to have zero dependency. Thanks to this constraint, I learned how to write a text editor, how to build a template engine, and how to handle DOM diffing for state-based rendering. I built a testing framework from scratch. And built a custom XML parser for handling RSS feed.

As a designer, I started experimenting with various design languages, from console-inspired base64 theming system to modern minimalist material design.
