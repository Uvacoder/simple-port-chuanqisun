3Dconnexion SpaceMouse Driver for Figma

Allow direct manipulation of the 2D canvas using the 3Dconnexion SpaceMouse hardware

Design

The tool went throught several iterations to fine-tuning the sensitivity and activation threshold. It also designed to account for mental models: manipulating canvas vs. manipulating camera.

Engineering

Performance is key in drivers. The project uses web animation API to reduce trashing and hand coded the entire UI and data process logic to reduce memory and CPU load. The logic is inspired by the "update loop" model in video games and real time systems.

Accessifiers

A non-profit service to help people discover accessible apps.

Engineering

To align with the project's goal and account for the frequent rotation of volunteer developers, I emphasized on the learning curve and accessibility of the tech stack itself. By using plain CSS, instead of CSS-in-JS and semantic HTML, instead of off the shelf UI toolkits, the project allows for easy onboarding and cross-contribution between designers who can code.

Switchboard

A tool for Microsoft Dynamics 365 teammates to quickly access demon environments

Design

Dynamics 365 is a B2B platform that creates apps on-demand for each business. This made it difficult for the designers to access the environment. The tool provides a one-click solution to allow instant access to demo environments. The project itself also embodies the progression of HCI paradigms. V1 was CLI that converses with the user. V2 was a desktop app that user to install and interact with. V3 is currently a cloud service that provides what user needs just in time.

Engineering

The app repurposed browser automation testing tool to become a co-pilot of human users, allowing instant typing of sign-in credentials. This was create before RPA became cool. The tool also went through several replatforms, allowing me to experiment and pick up new frameworks along the journey.
